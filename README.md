# Omea Fertility

**Votre alliée bienveillante à chaque étape de votre parcours PMA**

Application mobile iOS & Android d'accompagnement des femmes dans leur parcours de Procréation Médicalement Assistée.

- **Site** : https://jordantete.github.io/gaia-pma/
- **Repository** : https://github.com/jordantete/gaia-pma

## Fonctionnalités

- **Suivi médical** — Centralisation des RDV, résultats et traitements
- **Rappels intelligents** — Notifications pour traitements et rendez-vous
- **Communauté** — Espace d'échange entre femmes en parcours PMA
- **Ressources éducatives** — Articles validés par des professionnels de santé

## Stack technique

- HTML5 / CSS3 / JavaScript (Vanilla)
- Hébergement : GitHub Pages
- Déploiement automatique sur push `main`
